# 🧪 测试目录

本目录包含项目的测试文件。

## 测试类型

### 回归测试
- **`test_date_fix.py`**: 日期解析修复回归测试（v1.1.2）
  - 测试相对时间表达（"最近两个月"、"近一周"等）
  - 验证 LLM 正确使用 `days` 参数
  - 验证日期计算基于当前日期而非训练截止日期

## 运行测试

```bash
# 运行单个测试
python3 tests/test_date_fix.py

# 运行所有测试（未来）
python3 -m pytest tests/
```

## 测试规范

1. **命名**: `test_<feature_name>.py`
2. **文档**: 每个测试文件需要文档字符串说明测试目的
3. **独立性**: 测试应该能独立运行，不依赖其他测试
4. **清理**: 测试后清理临时文件（如需要）
